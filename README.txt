This is a very simple demo project, which demonstrates the simplest examples of Java Servlet, JSP and Spring MVC.

To create a WAR file, from project root execute:
mvn package

To run application on Jetty server, from project root execute:
mvn jetty:run

To access application open the following URL in a browser:
http://localhost:8080/servlet_jsp_demo/

To create settings for Eclipse, from project root execute:
mvn eclipse:eclipse
